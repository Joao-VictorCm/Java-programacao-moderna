# Chapter 04 — Estruturas Repetitivas

Este capítulo aborda as **estruturas de repetição em Java**, utilizadas
para executar blocos de código múltiplas vezes com base em condições.

## Conteúdos abordados

- Estrutura `while`
- Estrutura `do / while`
- Estrutura `for`
- Controle de fluxo com `break` e `continue`
- Exercícios práticos com laços de repetição

## Arquivos do capítulo

- **WhileLoopExample.java**  
  Repetição controlada por condição utilizando `while`.

- **DoWhileLoopExample.java**  
  Estrutura `do / while`, garantindo ao menos uma execução.

- **ForLoopExample.java**  
  Laço `for` com controle por contador.

- **LoopControlBreakContinue.java**  
  Uso de `break` e `continue` para controle do fluxo dos loops.

- **LoopExercises.java**  
  Exercícios práticos aplicando estruturas repetitivas.

## Objetivo do capítulo

Entender como automatizar repetições em Java, utilizando
estruturas de laço para percorrer dados e resolver problemas
de forma eficiente.
