# Chapter 06 — Classes, Atributos, Métodos e Static

Este capítulo aborda os conceitos fundamentais da **Programação Orientada a Objetos em Java**.

Aqui são apresentados exemplos práticos de como modelar classes,
definir atributos, criar métodos e utilizar membros estáticos.

## Conteúdo do capítulo

- **Atributos**  
  Representação de dados de um objeto.

- **Métodos**  
  Definição de comportamentos e operações sobre os dados.

- **Membros estáticos (`static`)**  
  Elementos que pertencem à classe e não a uma instância específica.

## Objetivo

Compreender a estrutura básica de classes em Java e como organizar
dados e comportamentos de forma clara e reutilizável,
preparando o terreno para conceitos como construtores,
encapsulamento e herança.
