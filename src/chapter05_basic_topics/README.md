# Chapter 05 — Outros Tópicos Básicos em Java

Este capítulo aborda **tópicos fundamentais da linguagem Java**
que complementam a base da lógica de programação e são amplamente
utilizados no dia a dia do desenvolvimento.

## Conteúdos abordados

- Operadores bit a bit
- Manipulação de strings
- Funções matemáticas
- Classes wrapper e conceitos de boxing e unboxing

## Arquivos do capítulo

- **BitwiseOperators.java**  
  Uso de operadores bit a bit (`&`, `|`, `^`, `~`, `<<`, `>>`).

- **StringFunctions.java**  
  Principais métodos da classe `String` para manipulação de textos.

- **MathFunctions.java**  
  Uso da classe `Math` para operações matemáticas comuns.

- **WrapperClasses.java**  
  Classes wrapper, boxing e unboxing de tipos primitivos.

## Objetivo do capítulo

Consolidar o conhecimento de recursos essenciais da linguagem Java,
fornecendo ferramentas importantes para a manipulação de dados,
operações matemáticas e trabalho com objetos.
