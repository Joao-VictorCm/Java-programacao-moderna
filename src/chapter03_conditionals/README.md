# Chapter 03 — Estruturas Condicionais

Este capítulo aborda as **estruturas condicionais em Java**, permitindo
que o programa tome decisões com base em expressões lógicas e comparações.

## Conteúdos abordados

- Estrutura `if / else`
- Estrutura `switch / case`
- Operador ternário
- Aplicação prática de decisões condicionais

## Arquivos do capítulo

- **IfElseExample.java**  
  Uso da estrutura `if / else` para controle de fluxo.

- **SwitchCaseExample.java**  
  Controle de fluxo com múltiplas opções utilizando `switch / case`.

- **TernaryOperatorExample.java**  
  Uso do operador ternário como forma simplificada de condição.
