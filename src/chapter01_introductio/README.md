# Chapter 01 — Introdução à Linguagem Java

Este capítulo apresenta os primeiros conceitos da linguagem Java,
servindo como base para todo o restante do curso.

## Conteúdos abordados

- Estrutura básica de um programa Java
- Método `main`
- Tipos primitivos
- Declaração de variáveis
- Entrada e saída de dados no console

## Arquivos do capítulo

- **HelloWorld.java**  
  Primeiro programa em Java, utilizando `System.out.println`.

- **JavaStructure.java**  
  Estrutura de um programa Java (classe, método `main`, organização).

- **PrimitiveTypes.java**  
  Exemplos dos principais tipos primitivos (`int`, `double`, `char`, `boolean`, etc.).

- **Variables.java**  
  Declaração, inicialização e uso de variáveis.

- **InputOutput.java**  
  Entrada de dados com `Scanner` e saída no console.

## Objetivo do capítulo

Compreender a sintaxe inicial da linguagem Java e se familiarizar
com a estrutura de programas simples, preparando o terreno
para estruturas sequenciais e lógicas mais avançadas.
