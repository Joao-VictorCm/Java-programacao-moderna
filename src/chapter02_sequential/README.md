# Chapter 02 — Estrutura Sequencial

Este capítulo aborda a **estrutura sequencial em Java**, onde as instruções
são executadas em ordem, sem desvios de fluxo, formando a base da lógica de programação.

## Conteúdos abordados

- Operadores aritméticos
- Operadores de atribuição
- Conversão de tipos (casting)
- Saída de dados formatada
- Resolução de exercícios sequenciais

## Arquivos do capítulo

- **AssignmentOperators.java**  
  Uso de operadores de atribuição simples e compostos.

- **ArithmeticOperators.java**  
  Operadores aritméticos básicos (`+`, `-`, `*`, `/`, `%`).

- **CastingExample.java**  
  Conversão explícita e implícita de tipos primitivos.

- **FormattedOutput.java**  
  Formatação de saída utilizando `printf` e `String.format`.

- **SequentialExercises.java**  
  Exercícios práticos aplicando a estrutura sequencial.

## Objetivo do capítulo

Desenvolver a base da lógica de programa
